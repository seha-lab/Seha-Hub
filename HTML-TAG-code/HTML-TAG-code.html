세하 마크업 변환기

입력창에 세하 문법으로 글을 쓰고 변환하기를 누르면, 아래에 미리보기와 HTML 결과가 생성됩니다.

사용법

1. 아래 입력칸에 글을 작성합니다.
2. 필요한 부분에 문법을 넣습니다.
3. 변환하기를 누릅니다.
4. 아래 미리보기에서 확인하거나 HTML 복사로 결과를 가져갑니다.

문법

&&텍스트&& 굵게
%%텍스트%% 형광펜
///텍스트/// 기울임
^^텍스트^^(255,120,120,0.85) 색상
***2*** 아래첨자
**2** 위첨자
()() 줄바꿈
$$세하$$ 좌우 중앙선

예시

&&중요&& %%강조%% ///분위기///
물은 H***2***O
E = mc**2**
첫 줄()()둘째 줄

참고: ***는 아래첨자, **는 위첨자입니다. 입력할 때 별 개수를 헷갈리지 않게 써주세요.

입력

미리보기

변환된 HTML
